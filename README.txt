<h1 align="center"> Sistema de Proejetos (POO JAVA) </h1>

Funções implementadas:
- Login
- Cadastro de usuário
- Relatórios do projeto
(Além de possui todas as informaçoes necessárias para criaçao do projeto e suas atividades, como:
  - Identificação
  - Descrição
  - Data e hora de início
  - Data e hora de término
  - Nome do coordenador
  - Profissionais envolvidos
  - Atividades (e seus atributos))

Padrões e Exceções:
- Singleton Design Pattern
- InputMismatchException 

Durante o funcionamento do programa, os dados de entrada são devidamente validados para que a entrada seja compatível com o tipo esperado.


No projeto foram implementadas as funções de login, cadastro de novo usuario, registro de projeto e relatório de todos
os projetos registrados no sistema. No projeto também foram implementados o design pattern "Singleton"
e a exceção "InputMismatchException" da biblioteca java.util. Os dados de entrada são devidamente validados.
